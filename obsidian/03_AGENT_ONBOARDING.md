## Agent rules

1. Do not invent canon.
2. Do not change calculation formulas without approval.
3. Do not expose secrets.
4. Do not add new product modes before demand validation.
5. Do not automate before manual sales prove demand.
6. Do not confuse Personal Myth with therapy.
7. Do not promise predictions, healing, or guaranteed outcomes.

## First safe package for future agents

The first safe package should usually be one of:

- documentation cleanup;
- prompt QA;
- Big Research manual template;
- Telegram message copy improvement;
- lead flow inspection;
- domain/HTTPS verification;
- smoke-test checklist.

## Production warning

The Telegram bot is already active. Treat it as production.

Any change to bot runtime requires:

1. backup;
2. syntax check;
3. clear rollback path;
4. explicit approval;
5. post-restart log check.
