Zerkalo / Цифровой Код

Политика обработки персональных данных

Редакция для первичного smoke-test запуска. Перед масштабной рекламой и подключением оплаты документ нужно дополнительно проверить юридически.

1. Какие данные собираются

Сервис может собирать данные, которые пользователь добровольно указывает в формах заявки:

  • имя;
  • дату рождения;
  • контакт для связи: Telegram, email или другой указанный способ;
  • короткий запрос или комментарий к будущему персональному разбору.

2. Зачем используются данные

Данные используются только для подготовки персонального разбора, связи с пользователем, уточнения запроса и ручной обработки заявки на «Большое исследование».

3. Где хранятся данные

На текущем smoke-test этапе заявки сохраняются на сервере проекта в локальном файле leads.json. Данные не публикуются в репозитории и не предназначены для открытого доступа.

4. Передача третьим лицам

Данные не продаются и не передаются третьим лицам для рекламы. Техническая обработка может осуществляться через инфраструктуру хостинга и серверные сервисы, необходимые для работы сайта.

5. Срок хранения и удаление данных

Пользователь может запросить удаление своих данных, написав владельцу проекта через контакт, по которому велась переписка после отправки заявки, либо через публичный контакт проекта после его публикации.

6. Границы продукта

«Цифровой Код» и «Личный Миф» являются информационно-аналитическими и образно-рефлексивными форматами. Сервис не является медицинской, психологической, финансовой или юридической услугой, не ставит диагнозы, не даёт предсказаний и не гарантирует конкретные события.

7. Согласие пользователя

Отправляя форму заявки и отмечая согласие с этой политикой, пользователь подтверждает, что добровольно передаёт указанные данные для связи и подготовки персонального разбора.
