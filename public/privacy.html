Политика обработки персональных данных

Эта редакция описывает фактическую обработку данных в текущей версии сайта и Telegram-сервиса. Перед подключением оплаты документ подлежит отдельной юридической проверке.

1. Какие данные собираются

Сервис может собирать данные, которые пользователь добровольно указывает в формах заявки:

  • имя;
  • дату рождения;
  • контакт для связи: Telegram, email или другой указанный способ;
  • короткий запрос или комментарий к будущему персональному разбору.

2. Зачем используются данные

Данные используются для расчёта Цифрового паспорта, подготовки персонального разбора, уточнения вопроса пользователя, доставки результата и связи по оставленной заявке.

3. Где хранятся данные

Заявки сайта сохраняются на сервере проекта. Данные Telegram-сценария и служебные статусы отчётов хранятся в закрытом хранилище сервиса. Они не публикуются в репозитории и не предназначены для открытого доступа.

4. Внешние технические сервисы

Данные не продаются и не передаются для рекламы. Для работы сервиса используется инфраструктура хостинга, Telegram и внешний провайдер языковой модели. При подготовке персонального текста провайдеру могут передаваться дата рождения, рассчитанная цифровая формула, выбранная тема, текст вопроса и ответы на уточнения. Это необходимо для генерации контекстного текста. Контактные данные из формы заявки не используются для обучения публичной модели со стороны проекта.

Перед отправкой запроса пользователь вправе не указывать сведения, которые считает чувствительными. Для разбора достаточно описать ситуацию без фамилий, адресов, документов и данных третьих лиц.

5. Срок хранения и удаление данных

Пользователь может запросить удаление своих данных, написав владельцу проекта через контакт, по которому велась переписка после отправки заявки, либо через публичный контакт проекта после его публикации.

6. Границы продукта

«Цифровой Код» и «Личный Миф» являются информационно-аналитическими и образно-рефлексивными форматами. Числовые трактовки являются рабочими гипотезами для самонаблюдения. Сервис не является медицинской, психологической, финансовой или юридической услугой, не ставит диагнозы, не даёт предсказаний и не гарантирует конкретные события.

7. Согласие пользователя

Отправляя форму заявки и отмечая согласие с этой политикой, пользователь подтверждает, что добровольно передаёт указанные данные для связи и подготовки персонального разбора.
