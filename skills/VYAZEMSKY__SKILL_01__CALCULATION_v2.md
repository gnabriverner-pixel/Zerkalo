Вызывать при ЛЮБОМ расчёте. Это фундамент системы "Цифровой код".

1. ПЯТЬ ГЛАВНЫХ ЧИСЕЛ (АЛГОРИТМ РАСЧЁТА)

В системе "Цифровой код" мы опираемся на 5 главных чисел. Крайне важно фиксировать не только однозначное (главное) число, но и СОСТАВНОЕ, из которого оно получилось.

1. ЧИСЛО ДУШИ (ЧДш)
Формула: День рождения.
Пример: 6-е число -> 6. 29-е число -> 29/11/2.

2. ЧИСЛО ПУТИ (ЧП)
Формула: Сумма ВСЕХ цифр даты рождения.
Пример: 06.05.1986 -> 6+5+1+9+8+6 = 35. Главное число: 3+5 = 8. Записываем: 35/8.

3. ЧИСЛО НАПРАВЛЕНИЯ (ЧН)
Формула: Число Души (базовое) + Число Пути (составное).
Пример: ЧДш = 6. ЧП = 35. ЧН = 6 + 35 = 41. Главное число: 4+1 = 5. Записываем: 41/5.

4. ЧИСЛО ВЫРАЖЕНИЯ (ЧВ)
Формула: День рождения + Месяц рождения.
Значение: Как человек воспринимает мир и какие инструменты ему даны для реализации на его пути.
Пример: 06.05 -> 6 + 5 = 11. Главное число: 1+1 = 2. Записываем: 11/2.

5. ЧИСЛО РЕЗУЛЬТАТА (ЧРз)
Формула: Число Души (базовое) + Число Пути (составное) + Число Направления (составное).
Пример: ЧДш = 6, ЧП = 35, ЧН = 41. ЧРз = 6 + 35 + 41 = 82. Главное число: 8+2 = 10 -> 1. Записываем: 82/10/1.

2. ПОРЯДОК РАСЧЁТОВ ДЛЯ ФАЙЛА
1. ЧДш (Число Души)
2. ЧП (Число Пути)
3. ЧН (Число Направления)
4. ЧВ (Число Выражения)
5. ЧРз (Число Результата)
6. Матрица (простая и детальная)
7. ЛГ (Личный Год)
8. Денежный код (ЧДш - ЧП - ЧН - ЧРз)

3. МАТРИЦА (ДЕТАЛЬНАЯ)
К цифрам даты добавляются рабочие числа (РЧ).
- РЧ1 = сумма всех цифр даты (это и есть составное Число Пути).
- РЧ2 = сумма цифр РЧ1.
- РЧ3 = РЧ1 минус (первая ЗНАЧАЩАЯ, то есть не равная нулю, цифра дня рождения × 2).
- РЧ4 = сумма цифр РЧ3.
Все цифры РЧ1–РЧ4 добавляются в матрицу.

4. МАСТЕР-ЧИСЛА И СОСТАВНЫЕ
Мастер-числа: 11, 22, 33. Не приводятся к однозначному — записываются как двойное (11/2, 22/4, 33/6).
Составные числа (35, 41, 82 и т.д.) ОБЯЗАТЕЛЬНО раскрываются при анализе. Они показывают, ИЗ ЧЕГО состоит главное число.
