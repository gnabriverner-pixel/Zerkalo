# Zerkalo / «Цифровой Код»

Это премиум-интерфейс инструмента самопознания по системе «Цифровой Код» (развитие системы Канон). Приложение анализирует дату рождения, выстраивает персональную архитектуру (Матрицу и Пять главных чисел) и генерирует уникальное «Первое зеркало» — базовый разбор сильных и слабых сторон личности.

## Как это устроено

### Interpretation Core
Математическое ядро (`calculator.ts`) переводит дату рождения в 5 главных и 35 составных чисел. Аналитический модуль (`numberKnowledge.ts`, `compoundKnowledge.ts`) сопоставляет эти числа с архетипами, планетами и ключевыми энергиями.

### First Mirror (Первое зеркало)
Главный вау-эффект приложения. Пользователь вводит дату и получает структурированный текст, содержащий: 
- Главный узор (синтез чисел Души и Пути)
- Светлые стороны и зоны напряжения
- Первый практический шаг
Интерфейс выполнен в стиле *modern premium editorial* с элементами античной лаконичности (цвета мрамора, цвета тёплой бумаги, античное золото, шрифты с засечками).

### Big Research Teaser & Lead Capture
В конце «Первого зеркала» пользователю предлагается «Большое исследование» — закрытый премиальный продукт. Нажатие на кнопку вызывает модальное окно заявки (имя, контакт, запрос). Заявка отправляется на `/api/lead` и собирается в локальный (или внешний) список.

### Локальная детерминированная генерация
Если API недоступно, приложение использует `deterministicMirror` из `interpretation.ts`, собирая идеальное структурное Первое Зеркало на основе базы знаний без помощи нейросетей. 

### Gemini как опциональный энхансер
При наличии `GEMINI_API_KEY` (на сервере Node + Express) система отправляет детерминированный каркас и жесткие safety-правила в LLM (Gemini). Модель стилизует текст под глубокий, премиальный, авторский слог (без магии и воды), возвращая готовый JSON объект `FirstMirror`.

### Safety & Ethical Boundaries
Важное правило продукта: **Проект использует аналитико-образный язык, но не является терапией, диагностикой, предсказанием или духовной практикой.**
Никаких диагнозов, никакой фатальности, никакой магии. Мы очистили проект от эзотерического и тревожного регистра. Мы не используем слова вроде «судьба неизбежна», «терапия», «кармический сбой», «исцеление», «высшие энергии». Роль приложения — структурно подсветить потенциал и напряжение, а выбор всегда остается за пользователем.

## Personal Myth / Сказка про тебя

«Личный миф» — образно-рефлексивный слой проекта. Он помогает увидеть текущее состояние через метафорическую историю, главный образ, внутреннее напряжение, скрытый ресурс и один мягкий следующий шаг.

Это не диагностика, не инструкция к действию и не замена живой поддержке.

```bash
npm install

# Для использования ИИ-генерации текста (опционально)
cp .env.example .env
# Вставьте GEMINI_API_KEY в .env

# Запуск dev-сервера (Frontend + Backend на порту 3000)
npm run dev
```

## Тестирование (How to Test)
Проект покрыт unit-тестами с помощью **Vitest**. Проверяется математика калькулятора, корректность структуры Первого Зеркала и наличие/отсутствие запрещенных слов в словаре системы.

```bash
npm run test
```
