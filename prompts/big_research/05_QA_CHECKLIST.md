# 05_QA_CHECKLIST — Big Research Prompt Pack

Use this checklist after generating or drafting a Big Research document.

## 1. Calculation

- [ ] Date is correct.
- [ ] Formula is correct.
- [ ] Root numbers are correct.
- [ ] Compound numbers are preserved.
- [ ] Matrix is correct.
- [ ] Current period / personal year is correct if used.

## 2. Source grounding

- [ ] Every major claim is tied to a number, compound, matrix, period, or user-provided context.
- [ ] Hypotheses are not presented as canon.
- [ ] No external text is copied verbatim.
- [ ] The report is original in wording.

## 3. Structure

- [ ] The document has a clear opening.
- [ ] The numeric passport is visible.
- [ ] The main pattern is synthesized.
- [ ] Each main number is explained by position.
- [ ] Compound numbers are explained.
- [ ] Strengths and tension points are clear.
- [ ] Practical route is included.
- [ ] Closing Living Word is personal.

## 4. Language quality

- [ ] No generic AI praise.
- [ ] No repeated fillers.
- [ ] No awkward grammar.
- [ ] No excessive abstraction.
- [ ] No overuse of “deep”, “strong”, “inner”.
- [ ] No bureaucratic phrasing.

## 5. Safety

- [ ] No diagnosis.
- [ ] No treatment advice.
- [ ] No predictions.
- [ ] No fatalistic claims.
- [ ] No guaranteed outcomes.
- [ ] No medical / legal / financial instruction.
- [ ] Boundary disclaimer is present.

## 6. Product value

- [ ] The difference from the free First Mirror is obvious.
- [ ] The client receives a useful route, not only a portrait.
- [ ] The report creates trust for follow-up.
- [ ] The reader understands what to do next.

## 7. Delivery

- [ ] Client name is correct.
- [ ] Date is correct.
- [ ] File name is clean.
- [ ] PDF opens.
- [ ] Mobile readability checked if sending through Telegram.
- [ ] No internal comments or placeholders remain.
