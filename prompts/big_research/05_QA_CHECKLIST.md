# 05_QA_CHECKLIST — Чеклист качества для Большого исследования

---

Этот чеклист — сокращённая версия `docs/REPORT_QA_GATE.md`, адаптированная для быстрой проверки после генерации.

## Быстрая проверка (7 пунктов)

1. [ ] Нет ли фраз, которые подойдут почти любому человеку?
2. [ ] Есть ли в каждой секции явная опора на конкретные числа из расчёта?
3. [ ] Распакованы ли составные числа там, где это требуется?
4. [ ] Не повторяется ли одна мысль разными словами в разных секциях?
5. [ ] Звучит ли как живой эксперт, а не как безопасная AI-модель?
6. [ ] Числовой паспорт — сухой и чистый?
7. [ ] Нет ли дешёвого сленга, псевдопоэзии и 3-го лица?

## Красные флаги (если найден хотя бы один — переделка)

- [ ] Фраза «это говорит о том, что» встречается 2+ раз.
- [ ] Число не соответствует расчёту (ошибка в числах/планетах).
- [ ] Медицинский, финансовый или экстремальный совет.
- [ ] Нет практического маршрута.
- [ ] Документ ощущается как generic-шаблон.
