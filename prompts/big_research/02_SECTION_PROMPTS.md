# 02_SECTION_PROMPTS — Big Research Prompt Pack

Use these section prompts when generating a Big Research document.

## 1. Cover

Create a clean premium cover block with:

- client name;
- date of birth;
- formula;
- document title;
- short subtitle;
- date prepared.

Do not use mystical or predictive copy.

## 2. How to read this document

Explain the three-layer structure:

1. calculation skeleton;
2. canon-based meaning;
3. living interpretation.

Make clear: this is not a verdict, not a prediction, not a diagnosis.

## 3. Numeric passport

Show the formula in a concise table.

For each main number:

- position;
- compound;
- root;
- planet / symbolic energy;
- one-line living meaning.

## 4. Main pattern

Synthesize all main positions into a single portrait.

Required movement:

```text
inner core -> action style -> realization route -> mature outcome -> central tension.
```

## 5. Mind / Soul number

Write:

- what this number means in the inner core;
- how the compound modifies it;
- how it appears in daily life;
- what becomes strength;
- what becomes overuse;
- one practical recommendation.

## 6. Action / Path number

Write:

- how the person acts;
- what makes them effective;
- what blocks action;
- what rhythm or behavior helps.

## 7. Expression number

Use only if present.

Write:

- how the person enters a field;
- what others may notice first;
- where this helps;
- where it may create a mask or misunderstanding.

## 8. Realization / Direction number

Write:

- where energy should be directed;
- what type of work or contribution fits;
- how compound numbers deepen the route;
- what should be tested in real life.

## 9. Outcome / Result number

Write:

- what mature assembly looks like;
- what quality life asks the person to develop;
- what false version of maturity to avoid.

## 10. Compound numbers

For each compound:

```text
number -> digits -> root -> transition formula -> accent -> risk -> recommendation.
```

Avoid generic “energy X and Y create Z” without concrete interpretation.

## 11. Matrix

Write:

- what is repeated;
- what is present once;
- what is absent;
- what lines are active;
- what requires conscious construction.

Do not frame missing digits as defects.

## 12. Strengths

List 3–5 strengths grounded in the formula.

Each strength:

- name;
- source in calculation;
- life expression;
- practical use.

## 13. Tension points

List 3–5 tensions grounded in the formula.

Each tension:

- name;
- source in calculation;
- how it appears;
- what form it asks for.

## 14. Money and realization

Write in practical language:

- best value creation mode;
- suitable work arenas;
- money leaks;
- experiment for the next 30 days.

Do not promise income.

## 15. Relationships

Write:

- contact style;
- what supports closeness;
- what creates friction;
- how to speak/act more consciously.

Do not promise romantic outcomes.

## 16. Current period

If personal year is present:

- number and planet;
- year theme;
- interaction with formula;
- useful focus;
- what not to overdo.

## 17. 30-day route

Give:

- 3 support points;
- 3 tension points;
- 3 actions;
- one main step.

The main step must be simple enough to start today.

## 18. Living Word

Close with a direct, personal, non-generic address.

Structure:

```text
central recognition -> old pattern -> new form -> final line.
```

## 19. What to do next

Offer:

- reread one section;
- choose one step;
- ask one follow-up question;
- book live consultation if appropriate.
